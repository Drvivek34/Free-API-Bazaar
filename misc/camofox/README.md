# Camofox

Open-source, headless anti-detection browser automation server designed for AI agents. Free to run and self-host locally or on own infrastructure (Docker/VPS) using the open-source Camoufox Firefox fork. Integrates with agents via local base URL overrides. Supports access security via custom CAMOFOX_API_KEY authentication tokens.
