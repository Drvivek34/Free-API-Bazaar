# 🆓 Free API Bazaar

> A categorized, community-maintained directory of **free LLM & AI APIs** — who gives what away, with what limits.

[![Mega AI Bazaar](https://img.shields.io/badge/🌐_Mega_AI_Bazaar-browse_all-6C5CE7)](https://drvivek34.github.io/Mega-AI-Bazaar/) [![Submit](https://img.shields.io/badge/➕_Submit-a_free_API-2ecc71)](https://github.com/Drvivek34/Free-API-Bazaar/issues/new/choose) [![License](https://img.shields.io/badge/License-MIT-blue)](LICENSE)

Free API Bazaar (the bazaar formerly tracked as "awesome free LLM API resources") is a
living catalog of providers that offer **free or free-tier access** to LLMs, image, audio,
video, embeddings, and multimodal models — with their real limits, OpenAI-compatibility,
and signup links. The most generous picks are surfaced in **featured/**.

> ⏱️ **Provider terms change constantly.** Entries are dated; always verify before relying on a limit.
