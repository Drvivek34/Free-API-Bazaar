# Contributing to Free API Bazaar

Add a free API via the [submission form](https://github.com/Drvivek34/Free-API-Bazaar/issues/new/choose) or a PR.

## Layout
```
<category>/
└── <provider-name>/
    └── README.md   # free-tier limits, models, auth, OpenAI-compat, signup link, source
```

## Each entry must record
- Provider name + signup/docs URL
- Free-tier limits (RPM/RPD/tokens) and whether a card is required
- Models available
- OpenAI-compatible? (base_url if so)
- Source of the info + date verified

Provider terms change fast — keep entries dated. Don't post API keys.

Part of the [Mega AI Bazaar](https://drvivek34.github.io/Mega-AI-Bazaar/).
