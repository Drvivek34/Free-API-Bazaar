# Deepgram API

Billed by usage (per-second billing). New users receive $200 in one-time free credits (approx. 12,000 minutes of transcription). Startup program provides up to $100,000 in credits over 12 months. Paid plans: Pay-As-You-Go, Growth (discounted rates with annual commitment), and Enterprise (custom rates/self-hosting). STT rates start at ~$0.0077/min, TTS starts at ~$0.015/1k characters, Voice Agent is ~$4.50/hour.
