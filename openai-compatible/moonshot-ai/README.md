# Moonshot AI

Kimi chat interface is free for casual users with query volume and context limits. Pro features (Agent Swarm, Deep Research) are available via monthly subscriptions: Moderato (~$19/mo), Allegretto, and Vivace (up to $199/mo). Developer API is pay-as-you-go ($0.55-$0.95 per million input, $2.50-$4.00 per million output tokens) with automatic context caching. Open-weight versions of Kimi K2 models are free to self-host.
