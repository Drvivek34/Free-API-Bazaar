# Phala

Decentralized confidential computing network. Agent Contracts can be deployed and hosted for free with no gas fees (dstack SDK is free). For Phala Cloud (Confidential VMs/GPU TEEs), new verified users may receive $20 in free credits, after which computing resources are billed on a pay-as-you-go basis (per-minute billing for instance types and $0.000139/GB/hour for storage).
