# OpenInference

Offers hosted model inference. Historically provides free rate-limited access for certain models (such as Qwen, Kimi, and DeepSeek, e.g., 500 prompts per day) for academic and research purposes. Paid API access is based on token consumption (e.g., Gemma 4 31B at $0.06/M input and $0.35/M output tokens; GPT-OSS 120B at $0.03/M input and $0.15/M output tokens).
