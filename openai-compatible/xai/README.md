# xAI

Grok chatbot has a limited free tier (~10 messages every 2 hours) on grok.com/X app. The developer API (console.x.ai) offers $25 in promotional credits for new accounts. A data-sharing opt-in program offers up to $150/month in credits (requires minimum $5 spend to enroll). Paid API usage is consumption-based (e.g., Grok 4.3 at $1.25/M input and $2.50/M output tokens).
